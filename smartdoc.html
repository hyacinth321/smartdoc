Project Introduction
This is a smartdoc project. Specific operations are as follows: when running a corresponding Python file such as smartdoc.py and passing the corresponding srs.txt and code.py files to it, two HTML files will be automatically generated, which are srs.html and code.html, and the content of srs.txt is the srs of code.py. In the generated code.html file, there are not only corresponding code, but also strings in front of each block of code (with specific forms), which can link to another HTML file, that is, the specific location of srs.html. Correspondingly, in the srs.html file can also jump to the corresponding location of the code.html file.

Document

SmartDoc, a project which aim at making users to find the corresponding explanation in the srs file more quickly when thinking about the code.
The format of the command line is python smartdoc.py code.py srs.txt. The srs.txt is a plain text file and the code.py is a source code file.
Two html file will be generated which are code.html and srs.html.
In the srs.html, there will be links which can be in a sentence "{see ???}" to go the corresponding place in the code.html. And code.html also has links to go to srs.html for correct explanation.
